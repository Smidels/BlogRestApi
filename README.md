## BlogRestApi

``BlogRestApi`` is a simple Rest Api Blog.

Basic functions:
==========
*user singup
*user login
*post like
*post unlike
==========
### Start
```
> git clone  https://github.com/Smidels/BlogRestApi.git
> cd BlogRestApi
> python -m venv venv
> pip install -r requirements.txt
(venv)>python manage.py runserver
```
